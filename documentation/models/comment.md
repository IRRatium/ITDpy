## Модель Actor

### Пример ответа Actor
```json
"actor": {
	"id": "330dea20-bb7c-4c96-ad09-97150f1ad5f6",
	"displayName": "TM",
	"username": "FIRST_TM",
	"avatar": "™"
},
```
|Поле|Тип|Описание|
|--|--|--|
|id|str|ID пользователя|
|display_name|str|имя|
|username|str|Username|
|avatar|str|Emoji аватар|
